Specifications

  • First row with patient identifiers (first column with no value to use).
  • Second row with each patient's class, the dependent variable that classifies each participant in the sample (as in the previous case, the first column will be neglected).
  • The subsequent rows will take into account all the columns, the first representing the name of a gene and the later the values of genetic expression in each participant, respectively.
  • The separator used to distinguish each column from each row will be one of those indicated in the form field.
  • If there are filaments with repeated gene names, the platform is left with the first appearance of said gene, being the responsibility of the user this aspect, already being made an arithmetic mean of the repeated rows or any other desired method.
